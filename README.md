# NotkindBG Portfolio

Premium personal portfolio for Daksh Mungali (NotkindBG), built as a static React app with CDN React, Babel, Tailwind CSS, and custom CSS motion.

## Run Locally

Serve the folder:

```powershell
python -m http.server 5173
```

Then visit `http://localhost:5173`.

The page expects a `logo.png` file in this same directory. If it is missing, the UI falls back to a text mark.
